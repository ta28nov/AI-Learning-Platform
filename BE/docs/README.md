# Tài liệu hệ thống

Thư mục này gom các tài liệu kỹ thuật và báo cáo triển khai để dễ theo dõi theo từng nhóm:

- `architecture/`: kiến trúc tổng thể, baseline rewrite tài liệu.
- `reports/`: báo cáo mapping BE-FE, checklist demo/E2E.
- `plans/`: kế hoạch triển khai (tham chiếu).

## Danh mục hiện tại

- `architecture/SYSTEM_ARCHITECTURE.md`
- `architecture/DOCUMENTATION_REWRITE_BASELINE.md`
- `reports/BE_FE_CONTRACT_MATRIX.md`
- `reports/DEMO_E2E_CHECKLIST.md`

## Ghi chú

- `BE/schemas` và `BE/routers` vẫn là nguồn chuẩn khi đối soát contract.
- Tài liệu cũ ở root đã được chuyển về đây để giảm phân tán.
